# v0.6.0 章节缓存与翻页优化

## 新功能

- **章节文件缓存**：导入时按章节标题自动拆分，缓存到本机 `book-cache` 目录，章内翻页只加载当前章，速度明显提升
- **跨章翻页**：章末进入下一章；章首向上翻回到上一章最后一页

## 修复

- 摸鱼计算器阅读字数恢复**实时统计**（约每秒刷新）
- 点击阅读区恢复窗口聚焦，隐藏失焦时的 Windows 标题栏色带
- 滚轮翻页与顶栏交互回归正常

## 说明

- 无章节标题的书仍作为单章缓存，翻页速度与旧版相同
- 已导入的旧书下次打开时自动生成章节缓存
- 详见应用内「设置 → 说明书」与仓库 `KNOWN_ISSUES.md`
